# 📘 Formação TypeScript - Digital Innovation One (DIO)

Repositório dedicado a armazenar os projetos, desafios de código e anotações do bootcamp **Formação TypeScript** da [DIO](https://www.dio.me/). 

Este curso aborda desde os fundamentos do TypeScript, um superset do JavaScript mantido pela Microsoft, até a sua aplicação no Front-End com **React** e no Back-End com **Node.js** e **Express**.

## 🎯 Objetivo da Formação
Desenvolver aplicações robustas e seguras aplicando a tipagem estática do TypeScript. A trilha explora o paradigma da Programação Orientada a Objetos (POO), a criação de APIs RESTful e a construção de interfaces previsíveis e componentizadas com React, integrando o Back-End ao Front-End.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** TypeScript, JavaScript
* **Back-End:** Node.js, Express
* **Front-End:** React
* **Paradigma:** Programação Orientada a Objetos (POO)
* **Arquitetura:** REST API

## 📚 Tópicos Abordados

### 1. Fundamentos e Tipagem
* Vantagens do TypeScript sobre o JavaScript puro.
* Tipagem estática, inferência de tipos, Interfaces e Types.
* Mais segurança, previsibilidade e autocompletar no desenvolvimento.

### 2. Programação Orientada a Objetos (POO)
* **Classes e Objetos:** Como modelar o mundo real no código.
* **Abstração:** Criação de classes abstratas como moldes base (`abstract class`).
* **Encapsulamento:** Proteção de dados com modificadores de acesso (`public`, `private`, `protected`, `readonly`).
* **Herança:** Reaproveitamento de código entre classes mães e filhas (`extends`, `super`).
* **Polimorfismo:** Sobrescrita de métodos para comportamentos específicos.

### 3. Back-End Avançado com Express
* Boas práticas de código no ecossistema Node.js.
* Criação de servidores com Express.
* Desenvolvimento e estruturação de uma API REST.

### 4. Front-End com React + TypeScript
* Componentização utilizando TypeScript.
* Trabalhando com Propriedades (Props) tipadas.
* Integração eficiente entre Front-End e Back-End.
